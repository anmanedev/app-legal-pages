1. はじめに

「主治医メモ」は、日々の生活リズム・お薬の服用・通院の予定を簡単に記録し、診察のときに主治医へ見せられる形にまとめるためのiPhoneアプリです。毎日の記録は30秒ほどで入力でき、通院と通院の間の様子をレポートとして振り返ることができます。

大切な注意: 本アプリは記録を整理するためのものであり、診断・治療・服薬の指示など、医療上の判断を行うものではありません。体調に関する判断は、必ず主治医にご相談ください。

アプリには5つのタブがあります: ホーム・日々の記録・お薬の記録・通院の記録・レポート。

名称について: アプリ内の一部(通知の設定、レポートの一部の文言など)では、「日々の記録」機能のことを「生活メモ」と呼んでいる箇所があります。どちらも同じ機能を指します。

2. 初期設定の流れ

初めてアプリを開くと、次の順で初期設定が始まります。あとから設定画面の「通院先・お薬を編集」でいつでも見直せます。

  1. 音声入力を試す(任意) — マイクから日本語で入力する機能を短く試せます。「今はしない」を選んで飛ばすこともできます。
  2. 利用状況の送信 — アプリ改善のための匿名データ送信に同意するかを選びます(初期状態はOFF。記録の内容そのものは送信されません)。
  3. 通院先の登録 — 1件目の通院先(病院名・診療科・先生の名前)を登録します。2件目以降の追加はプレミアム機能です。
  4. お薬の登録 — お薬の名前と飲むタイミングを登録します。複数登録でき、「お薬なしで通院日へ」で飛ばすこともできます。
  5. 次回通院日の登録 — 次の通院予定日を登録します。
  6. 通知の設定 — お薬の記録時間や通院前の確認を、iPhoneの通知でお知らせするかを選びます。
  7. アプリロック(任意) — Face ID / Touch IDでアプリにロックをかけられます。

3. ホーム画面の見方

ホーム画面の上部には「今日のやること」が表示され、その下に状況に応じたカードが並びます。右上の歯車ボタンから設定画面を開けます。

今日のやること

その日の状態に応じて、最大3件の項目が表示されます。タップすると該当の画面へ移動します。

  • 「服用を記録する」 — まだ記録していないお薬がある場合
  • 「日々の記録を書く」 — 今日の記録がまだの場合
  • 「次回通院を登録する」 — 次の通院予定が未登録の場合

すべて完了すると「今日の記録はできています。」と表示されます。

カードの種類と表示条件

カード表示される条件・役割
明日は通院予定日です明日が通院予定日で、前日通知をONにしている場合。病院名と予定時刻を表示。「閉じる」で当日中は非表示にできます(翌日また条件を満たせば表示)。
今日のお薬まだ服用を記録していないお薬がある場合。「服用を記録する」でお薬の記録タブへ。
未入力日がありますアプリを使い始めた日以降の直近30日に、記録のない日がある場合。「確認する」で未入力日の一覧を開き、日付を選んでその日の記録を入力できます(過去日の入力はプレミアム機能)。

「通知は負担になっていませんか?」通知を使い始めてから約1週間後に一度表示され、「ちょうどいい」「多い」などを選んで通知の量を調整できます。

4. 日々の記録の使い方

1日の生活リズムを記録する画面です。すべての項目が任意で、入力できるところだけで大丈夫です。

記録の手順

  1. 画面下の「日々の記録」タブを開きます。
  2. 画面上部の「記録する日」を確認します(通常は今日)。過去の日付に記録したい場合は、日付の行をタップしてカレンダーから選びます(プレミアム機能。未来の日付は選べません)。
  3. 各項目を入力します(下記参照)。
  4. 画面下の「保存する」を押します。ホーム画面に戻り、「ホームに反映しました。」と表示されます。

入力項目(上から順)

  • 血圧・体重 — 「記録する」をONにすると、上下の血圧と体重を「+/−」ボタンで入力できます(体重は0.1kg単位)。前回の値が最初から表示されるので、変わったところだけ調整できます。記録しない日はOFFのまま保存できます。
  • 歩数 — キーボードで直接入力できます(最大6桁)。
  • 運動のハードさ — 通勤/徒歩/ジム/していない
  • 食事のリズム — 普段通り/遅め/不規則/1食抜き
  • 間食 — なし/少し/多め
  • 飲酒 — なし/少し/普通/多め
  • 体調メモ — 自由に入力できます。マイクボタンで音声入力もできます。
  • 食事メモを詳しく書く(折りたたみ) — 朝食・昼食・夕食のメモと、外食・コンビニの有無。
  • ヘルスケアの記録(折りたたみ) — 選んだ日の歩数・体重・血圧・Apple Watchの記録(ワークアウト・安静時心拍数)を確認できます。

ヘルスケア連携との関係

  • ヘルスケア連携(プレミアム機能)をONにしていると、血圧・体重・歩数はiPhoneの「ヘルスケア」アプリの当日の値が自動で表示されます。自動の値ではなく自分で入力したい場合は「手入力に切り替える」を押します。
  • 連携していない場合(無料でご利用の場合)も、血圧・体重・歩数はすべて手入力で記録できます。
  • 過去の日付に保存した場合、そのときのヘルスケアの値が「保存時点の記録」として固定され、あとから見ても同じ値が表示されます。

5. お薬の記録の使い方

服用を記録する

  1. 「お薬の記録」タブを開きます。登録したお薬がカードで並んでいます。
  2. 飲んだら「飲みました」を押します。服用時刻が記録され、ホーム画面に戻ります。
  3. あとで記録したい場合は「あとで」を押し、「15分後」「30分後」「1時間後」(再通知)または「今日は通知しない」を選びます。

服用の記録は1日ごとです。日付が変わると、カードは自動的に「まだ飲んでいません」の表示に戻ります。

お薬の追加・編集・削除

  1. 右上の「＋」でお薬を追加、カードの鉛筆ボタンで編集できます。
  2. お薬の名前(よく使う名前の候補も表示されます)と、飲むタイミング(朝食前後・昼食前後・夕食前後・毎週○曜日)を選びます。
  3. 通知を使う場合は「通知する」をONにして時刻を設定します。
  4. プレミアムをご利用の場合は、お薬を通院先ごとに分けて管理できます。
  5. 使わなくなったお薬は、編集画面の「お薬を一覧から外す」で一覧から外せます。

6. 通院の記録の使い方

「通院の記録」タブでは、通院先・次回の予定・主治医に聞いておきたいこと・診察後メモを管理できます。

通院先

  1. 「通院先」カードで、表示する通院先を切り替えられます(プレミアムで複数登録している場合)。
  2. 「選択中の通院先を編集」で病院名などを変更できます。
  3. 「通院先を追加」で新しい通院先を登録できます。無料では1件まで、プレミアムでは5件まで登録できます。

通院の予定

  1. 「予定」カードから次回通院日を登録・変更します。
  2. 日付・時間(未定でも登録可)・病院名・診療科を入力します。
  3. 事前のお知らせ通知(1週間前・前日・当日)をそれぞれON/OFFできます。

主治医に聞いておきたいこと

  1. 「主治医に聞いておきたいことを追加」を押します。
  2. カテゴリ(服薬・食事・飲酒・体調・その他)を選び、聞きたい内容を入力します。
  3. 登録した質問は一覧に表示され、編集・削除ができます。レポートにも表示されます。

診察後メモ

診察が終わったら、「診察で言われたこと」「次回までのメモ」「次回検査予定」「次回診察の準備(朝食なし・尿検査・採血・その他)」を記録できます。次回のレポート期間はこの通院日を起点に切り替わります。

7. レポート(主治医に見せるレポート)の使い方

「レポート」タブでは、前回の通院日の翌日から今日までの記録をまとめて確認できます。診察時にこの画面をそのまま主治医に見せることを想定しています。

  1. 画面上部で通院先を切り替えられます。通院当日は、その日に予約のある通院先が自動で選ばれます。
  2. 期間(例: 6月15日〜7月14日(30日間))を確認します。前回の通院日が見つからない場合は直近30日が表示されます。
  3. 「診察用まとめ」「ヘルスケアの記録」「主治医に聞いておきたいこと」「週ごとのまとめ」を順に確認します。
  4. 「日別の記録」を開くと、1日ずつの詳しい記録を確認できます。
  5. 画面下の「CSVを書き出す」で、期間内の全記録をCSVファイルとして共有できます(プレミアム機能)。
  6. 「診察後メモを記録する」から、その場で診察後メモを書けます。

ヘルスケアの記録の歩数は、手入力した値があればその値が優先して集計されます(手入力があれば無料でも歩数の集計が表示されます)。

8. 通知について

通知はすべてiPhoneの中で予約される「ローカル通知」で、サーバーから送られるものではありません。利用にはiOSの通知許可が必要です。

通知タイミング内容
生活メモ(日々の記録)のリマインダー設定した時刻に1日1回(初期値 21:00)。その日の記録を保存済みの場合は届きません。「今日の生活リズムを30秒で記録しませんか?」
お薬の通知お薬ごとに設定した時刻に毎日(毎週○曜日のお薬は毎週)「お薬の記録時間です」
お薬の再通知(あとで)「あとで」で選んだ15分後・30分後・1時間後「お薬の記録を確認しましょう」
通院・1週間前通院日の7日前の朝7:00「次回の通院が近づいています」
通院・前日通院日の前日の朝7:00「明日は通院日です」
通院・当日予約時刻の2時間前(時間未定の場合は朝7:00)「今日は通院日です」

通院の通知(1週間前・前日・当日)は、通院予定の編集画面でそれぞれON/OFFできます。

9. 設定画面の説明

ホーム画面右上の歯車ボタンから開きます。

  • プレミアム機能 — 機能の説明と購入画面への入口。購入済みの場合は「利用できます」と表示されます。
  • 通院先・お薬 — 「通院先・お薬を編集」から、初期設定と同じ流れで通院先・お薬・次回通院日を見直せます。
  • 基本設定 — 音声入力・利用状況の送信・通知の現在の状態を確認できます。
  • 通知設定 — 生活メモ通知のON/OFFと通知時刻(初期値21:00)を変更できます。お薬の通知はお薬の編集画面で、通院の通知は通院予定の編集で設定します。
  • アプリロック — Face ID / Touch IDによるロックのON/OFF。
  • 音声入力 — メモ欄での日本語音声入力のON/OFF。
  • ヘルスケア連携(プレミアム) — 歩数/Apple Watchの記録/体重/血圧の4つを個別にON/OFFできます(許可の流れは第12章のQ3参照)。データが反映されない場合の案内と「設定アプリを開く」ボタンもここにあります。
  • App Storeレビュー — 「このアプリをレビューする」。
  • フィードバック — 「ご意見・ご要望をメールする」でメールアプリが開きます。
  • ヘルプ・規約 — 利用マニュアル、プライバシーポリシーと特定商取引法に基づく表記をブラウザで開けます。
  • 利用状況の送信 — アプリ改善のための匿名データ送信のON/OFF(初期OFF)。お薬名・病院名・メモ本文・ヘルスケアの数値など、記録の内容そのものは送信されません。
  • 記録の保存 — 記録がこのiPhone内に保存されること(iCloudへの自動保存・同期は行わないこと)の説明。

10. プレミアム機能

プレミアム機能は買い切り(1回のお支払い)です。設定画面またはアプリ内の各案内から購入できます。機種変更などの際は、購入画面の「購入を復元」で復元できます。

機能無料プレミアム
通院先の登録1件5件まで
通院先ごとのお薬管理—(共通で管理)通院先別に分けて管理・表示
CSV出力—レポート期間の全記録を書き出し
ヘルスケア連携(自動読み取り)—(血圧・体重・歩数の手入力は無料でも可能)歩数・ワークアウト・心拍数・体重・血圧を自動表示
過去の日付の記録入力今日のみ過去の日付をさかのぼって入力・編集

購入後は、ホーム画面に「プレミアム機能へようこそ」などの案内カードが表示され、各機能をすぐに始められます。

11. データの保存・削除について

  • 保存場所 — すべての記録は、お使いのiPhoneの中に保存されます。アカウント登録は不要で、運営者のサーバーに記録が送られることはありません。iCloudへの自動保存・同期も行いません。
  • ヘルスケアの値 — 連携で読み取った歩数などは、原則としてアプリ内には保存せず、表示のたびにヘルスケアから読み直します(過去の日付に保存した記録と手入力値は例外として保存されます)。
  • CSV出力(プレミアム) — レポート画面の「CSVを書き出す」で、「主治医メモレポート_開始日_終了日.csv」というファイルを作成し、共有先(メール・ファイルなど)を選べます。共有後、アプリ内の一時ファイルは削除されます。
  • 個別の削除 — お薬は編集画面の「お薬を一覧から外す」で、質問メモは一覧から削除できます。日々の記録は同じ日付で上書き保存できます。
  • すべての記録の削除 — 現在のバージョンでは、アプリ内から全記録を一括削除する機能は提供していません。すべて削除したい場合は、iPhoneから本アプリを削除してください(通常、端末内のアプリデータも一緒に削除されます)。
  • 機種変更時のデータ引き継ぎ — 新しいiPhoneのセットアップ時に「クイックスタート」(古いiPhoneから直接データを移す機能)を使うか、iCloudバックアップまたはパソコン(Finder/iTunes)のバックアップから復元すると、本アプリの記録も一緒に新しい端末へ引き継がれます。プレミアム機能の購入情報は、同じAppleAccountでサインインしていれば、アプリを開いた時点で自動的に復元されます(万一復元されない場合は、プレミアム購入画面の「購入を復元」から手動で復元できます)。バックアップを使わずにまっさらな状態からセットアップした場合、記録は引き継がれませんので、機種変更の際は「クイックスタート」または「バックアップからの復元」を選ぶことをおすすめします。

12. よくある質問

Q1. 初期設定で通院先を1件しか登録できませんでした。あとから追加できますか?

初期設定では1件目のみ登録します。2件目以降は「通院の記録」タブの「通院先を追加」から登録できます(プレミアム機能・5件まで)。無料のままでも、1件目の通院先はいつでも編集できます。

Q2. 記録をぜんぶ消してやり直したいときは?

アプリ内に一括削除の機能はありません。iPhoneのホーム画面で本アプリを削除すると、端末内の記録も一緒に削除されます。削除した記録は元に戻せませんので、必要な記録は事前にCSVで書き出しておくことをおすすめします。

Q3. ヘルスケア連携をONにしたのに、歩数や体重が表示されません。

ヘルスケア連携の許可は2段階です: ①設定画面の「ヘルスケア連携」で読み取りたい項目のトグルをONにする → ②設定画面が閉じ、ホーム画面に表示されるカードのボタンを押すと、iOSの許可画面が表示されます。ここで「許可」を選ぶとONが確定します。
それでも反映されない場合は、iOSの許可画面で「許可しない」を選んでしまった可能性があります。iOSの仕様上、この画面は二度と表示されないため、iPhoneの「ヘルスケア」アプリ(またはiPhoneの設定アプリの本アプリのページ)からアクセス許可を変更してください。設定画面のヘルスケア連携の下部にある「設定アプリを開く」ボタンが近道です。

Q4. 昨日の記録を入れ忘れました。あとから入力できますか?

プレミアム機能「過去の日付の記録入力」で、過去の日付をさかのぼって入力できます。「日々の記録」画面上部の「記録する日」から日付を選ぶか、ホームの「未入力日があります」カード→一覧から日付を選んでください。未来の日付には入力できません。

Q5. 通知が届きません。

次の順でご確認ください: ①iPhoneの設定で本アプリの通知が許可されているか ②アプリの設定画面「通知設定」で生活メモ通知がONか(お薬はお薬の編集画面、通院は予定の編集で設定) ③生活メモのリマインダーは、その日の記録を保存済みの場合は届かない仕様です。
